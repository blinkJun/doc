# `CSS`选择器

## 基础选择器

| 选择器   | 别名       | 说明           | 版本 | 常用 |
| -------- | ---------- | -------------- | ---- | ---- |
| `tag`    | 标签选择器 | 指定类型的标签 | 1    | √    |
| `#id`    | ID 选择器  | 指定身份的标签 | 1    | √    |
| `.class` | 类选择器   | 指定类名的标签 | 1    | √    |
| `\*`     | 通配选择器 | 所有类型的标签 | 2    | √    |

| 选择器        | 别名           | 说明               | 版本 | 常用 |
